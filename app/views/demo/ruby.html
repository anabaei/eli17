Code Blocks

In iterators or loops everything between do ... end is code block. Variables define inside block are accessible only inside the block and we define it variable scope

Find

Go through of datas and if finds the block was true return the result

(1..10).find { |i| i % 3 == 0 } # returns 3, the first one on the list (1..10).include?(3) # among numbers 1 to 10 which one include 3 ?
